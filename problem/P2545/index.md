# P2545 P2545 [AHOI2004] 实验基地

## 题目描述

![](https://cdn.luogu.com.cn/upload/pic/1656.png)

## 输入格式

第一行有一个整数 $N(3 < N < 2000)$，表示登陆地带的大小是 $2 \times N$。随后的两行每一行有 $N$ 个整数（其绝对值不超过 $10^6$），表示对应的矩形土地的适用度评估值，各个整数之间用一个空格隔开。

## 输出格式

只有一行输出，为整数 $M$，即所确定的实验基地的适用度。

## 输入输出样例

### 样例 #1

#### 样例输入 #1

```
4
-1 2 -3 4
5 6 7 8
```

#### 样例输出 #1

```
31
```

## 说明/提示

数据范围见输入格式。
