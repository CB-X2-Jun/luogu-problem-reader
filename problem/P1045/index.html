题目描述

输出格式

第一行：十进制高精度数 $2^{P}-1$ 的位数。 第 $2\sim 11$ 行：十进制高精度数 $2^{P}-1$ 的最后 $500$ 位数字。（每行输出 $50$ 位，共输出 $10$ 行，不足 $500$ 位时高位补 $0$） 不必验证 $2^{P}-1$ 与 $P$ 是否为素数。
