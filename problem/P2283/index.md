# P2283 P2283 [HNOI2003] 多边形

## 题目描述

![](https://cdn.luogu.com.cn/upload/pic/1330.png)

## 输入格式

第一行一个正整数 $n\ (3 \le n \le 1500)$，表示多边形的顶点数。

之后 $n$ 行，每行两个整数 $x_i,y_i\ (-500 \le x_i,y_i \le 500)$，依次表示多边形的每个顶点的坐标。

## 输出格式

阴影部分的面积，保留 $2$ 位小数。

## 输入输出样例

### 样例 #1

#### 样例输入 #1

```
5									
0 0
0 6
6 6
6 0
3 3
```

#### 样例输出 #1

```
9.00
```
